# **App Name**: Academia Exam Platform

## Core Features:

- Student Exam Interface: React-based interface for students to take exams, featuring a timer, question navigation, and auto-save functionality.
- Question Management: Support for multiple question types including multiple choice, short text, and true/false questions.
- Real-Time Auto-Save and Indicators: Implement real-time auto-saving of answers with visual indicators to show save status. Notifications displayed via toast.
- Teacher Exam Creation: Dashboard for teachers to create new exams and upload questions.
- Admin User Management: Admin dashboard for managing users, including adding teachers and students.
- Result Display: Display the final score, correct and incorrect answers, and success rate on result page. This should also provide recommendations for student if they pass or fail the exam.
- Language Support: Application is able to switch the screen view based on if the content is using a LTR or RTL based language.

## Style Guidelines:

- Primary color: Deep blue (#3B82F6) to convey trust and professionalism.
- Background color: Light gray (#F9FAFB), a low saturation version of the primary to create a clean and modern look.
- Accent color: A lighter blue (#60A5FA), analogous to the primary, for interactive elements.
- Headline font: 'Poppins' (sans-serif) for headlines, providing a geometric and modern look.
- Body font: 'PT Sans' (sans-serif) for body text, balancing modernity with readability.
- Use consistent, professional icons to represent different actions and sections within the platform.
- Employ a clean, structured layout with clear distinctions between sections to ensure ease of navigation and use.
- Use subtle animations, such as fade-in effects and smooth transitions, to enhance user experience without being distracting.
